Que 1:Discuss the scope of var, let, and const

variables declared with var are in the function scope.
variables declared with let are in the block scope.
variables declared as const are in the block scope.

Que 2: Tell us the use cases of null and undefined

(Null)

A null means the absence of a value. you assign a null to a variable with the intention that currently this variable does not have any value but it will have later on.The type of null is the object.

(Undefined)

A variable is undefined when you haven't assigned any value yet , not even a null.
